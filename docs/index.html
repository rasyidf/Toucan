Universal i18n Resource Editor

A modern, open-source internationalization file editor for Windows — manage all your translations in one place.

Free & Open Source Windows Desktop MIT License 12+ File Formats

Features

Multi-Format Editing

Edit JSON, YAML, PO, RESX, Android XML, iOS .strings, XLIFF, Flutter ARB, CSV, TOML and more — all in one unified interface.

Side-by-Side Translation

View and edit source and target languages next to each other. Filter by untranslated, translated, and approved items.

AI Pre-Translation

Batch-translate using Google, DeepL, Microsoft, or OpenAI — with parameter preservation, formality settings, and preview before applying.

Fluent Design

Modern Windows UI with Mica backdrop, dark/light theme following your system preference.

Built for Scale

Handles large projects (10,000+ keys) with pagination, streaming JSON parser, and per-namespace operations.

Keyboard-First

Centralized keybinding service with 20+ configurable shortcuts. Cut/copy/paste, case conversion, and bulk actions.
